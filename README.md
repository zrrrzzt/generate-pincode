# generate-pincode

Node.js module for generating random pincodes.

## Installation

```sh
$ npm install generate-pincode
```
## Usage

```JavaScript
const gpc = require('generate-pincode')
const pin = gpc(4)

console.log(pin)

// => 1234
```
